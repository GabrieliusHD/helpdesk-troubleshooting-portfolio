# Ticket 002 – Printer Not Printing (Wi-Fi Configuration Fix)

## Issue Summary
User reported that the printer was connected to the computer and printer software was installed, but print jobs were not completing.

## Environment
- Windows PC
- Wireless printer
- Home Wi-Fi network

## Investigation Steps
1. Verified printer was powered on and detected by the operating system.
2. Confirmed printer software and drivers were installed correctly.
3. Ran built-in printer troubleshooting tool on Windows.
4. Checked printer queue – print jobs were stuck.
5. Verified printer connection status in printer software.

## Findings
- Printer software appeared functional.
- Printer was not properly connected to the Wi-Fi
