# Ticket 003: Wi-Fi Connected but No Internet Access

## Issue Summary
User reports device is connected to Wi-Fi but cannot access any websites or online services.

## Environment
- Device: Windows laptop
- Connection: Wi-Fi
- Location: Home network

## Questions Asked
- Are other devices able to connect to the internet?
- Does the Wi-Fi show “connected” or “limited”?
- Did this start suddenly or after a change?
- Has the device worked on this network before?

## Troubleshooting Steps
1. Verified Wi-Fi connection was active
2. Tested internet access on another device
3. Restarted Wi-Fi adapter on the laptop
4. Ran basic network troubleshooting in Windows
5. Restarted router and modem

## Findings
- Other devices regained internet after router restart
- Issue appeared to be related to local network connectivity rather than the device itself

## Resolution
Internet access restored after network equipment restart.

## Notes
This issue highlights the importance of verifying whether a problem is device-specific or network-wide before deeper troubleshooting.
