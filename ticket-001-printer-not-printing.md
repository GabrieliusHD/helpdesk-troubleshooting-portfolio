# Ticket 001: Printer Installed but Not Printing

## Issue Summary
User reports that the printer is installed on Windows and appears connected,
but documents are not printing.

## Environment
- Device: Windows PC
- Connection type: USB / Wi-Fi (unknown at initial contact)
- Printer software installed: Yes

## Initial Questions Asked
- Is the printer connected via USB or Wi-Fi?
- Does the printer appear online in Windows?
- Are there any error messages on the screen?
- Has this printer worked before?

## Troubleshooting Steps
1. Verified printer is installed in Windows
2. Checked printer status (not paused, set as default)
3. Attempted to print a test page
4. Restarted printer and PC
5. Ran Windows printer troubleshooter

## Result
Issue not resolved after basic software troubleshooting.

## Next Action
Proceed to check printer-side configuration and connectivity.

## Notes
This ticket demonstrates initial Tier 1 troubleshooting and issue isolation.
